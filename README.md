# Donations API

API REST desarrollada en .NET (C#) para la gestión de campañas de donación de sangre, como parte de un proyecto académico grupal.

## Tecnologías utilizadas

- .NET (C#)
- Entity Framework
- PostgreSQL
- Clean Architecture
- API REST
- Git

## Objetivo del proyecto

El objetivo de esta API es centralizar la lógica de negocio para la gestión de campañas de donación de sangre, permitiendo su consumo por aplicaciones frontend.

## Arquitectura

El proyecto sigue los principios de Clean Architecture, organizando el código en capas bien definidas:
- Domain
- Application
- Infrastructure
- Presentation (API)

## Estado actual

Este proyecto se encuentra en desarrollo.

Actualmente incluye:
- Estructura base de Clean Architecture.
- Modelado inicial de entidades.
- Configuración de acceso a datos con PostgreSQL.
- Endpoints iniciales de la API.
